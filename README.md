# Daily-Data-Science-Notebook
Daily notebooks for data science practices

### Notebooks Structure
- Day 1 / Day 3 : Introduction and Data Science Concepts
- Day 4 / Day 10 : Python (Introduction - Data Structures and Algorithms - OOP)
- Day 11 / Day 14 : R (Introduction-Graphs and Analysis)
- Day 15 / Day 18 : Data Analysis (Excel - Python - Dashboards with Power BI and Tableau - Data Visualization and Cleaning)
- Day 19 / Day 21 : Databases (Introduction - DDL and DML - Advanced SQL - Data Analysis with SQL)
- Day 21 / Day 28 : Mathematics (Linear Algebra - Probability and Statistics - Calculus - Operations Research)
- Day 28 / Day 40 : Artificial Intelligance (ML - Neural Networks - DL - Computer Vision)
- Day 41 / Day 43 : Physics for Data Scientists
- Day 44 / Day 47 : End-to-End Project
- Day 48 / Day 50 : Advanced (Linux - C++ - Docker - Web Development with HTML/CSS/JS or Frameworks like Quarto and Streamlit)

** Maybe the structure changes **
